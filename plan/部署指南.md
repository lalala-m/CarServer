## 🚀 部署步骤

### 步骤1：执行数据库脚本

**打开MySQL命令行或使用数据库管理工具（如Navicat），按顺序执行以下SQL：**

#### 1.1 创建store_admin表
```sql
-- 执行文件: plan/store_admin_table.sql
CREATE TABLE IF NOT EXISTS `store_admin` (
  `store_admin_id` int NOT NULL AUTO_INCREMENT COMMENT '4S店管理员ID',
  `user_id` int DEFAULT NULL COMMENT '用户ID',
  `store_id` int DEFAULT 0 COMMENT '所属4S店ID（0表示总部）',
  `employee_work_number` varchar(50) DEFAULT NULL COMMENT '员工工号',
  `employee_name` varchar(50) DEFAULT NULL COMMENT '员工姓名',
  `employee_gender` varchar(10) DEFAULT NULL COMMENT '员工性别',
  `contact_number` varchar(20) DEFAULT NULL COMMENT '联系号码',
  `examine_state` varchar(20) DEFAULT '待审核' COMMENT '审核状态',
  `user_state` varchar(20) DEFAULT '未启用' COMMENT '用户状态',
  `create_time` timestamp NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` timestamp NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (`store_admin_id`),
  UNIQUE KEY `uk_employee_work_number` (`employee_work_number`),
  KEY `idx_store_id` (`store_id`),
  KEY `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci COMMENT='4S店管理员表';
```

## 📖 使用说明

### 作为总部管理员（admin账号）

#### 1. 管理4S店信息管理员

1. 登录系统
2. 点击左侧菜单"系统用户" → "4S店管理员"
3. 可以：
   - **单个添加**：点击"添加"按钮
   - **批量导入**：下载导入文档模板，填写后点击"导入"上传
   - **批量审核**：勾选记录，点击"批量审核"
   - **批量启用**：勾选记录，点击"批量启用"

#### 2. 管理总部信息员

1. 点击左侧菜单"系统用户" → "信息员"
2. 可以：
   - **单个添加**：点击"添加"按钮
   - **批量导入**：下载导入文档模板，填写后点击"导入"上传

### 作为4S店管理员

#### 1. 管理业务顾问

1. 登录系统
2. 点击左侧菜单"系统用户" → "业务顾问"
3. 可以：
   - **单个添加**：点击"添加"按钮
   - **批量导入**：下载导入文档模板，填写后点击"导入"上传
   - **生成二维码**：点击操作列的"生成二维码"按钮

#### 2. 管理财务审核人员

1. 点击左侧菜单"系统用户" → "财务用户"
2. 可以：
   - **单个添加**：点击"添加"按钮
   - **批量导入**：下载导入文档模板，填写后点击"导入"上传

## 🔐 权限控制说明

### 总部管理员（admin）
- ✅ 可以查看所有门店的数据
- ✅ 可以管理所有4S店管理员
- ✅ 可以审核/启用任何门店的人员

### 4S店管理员
- ⚠️ 只能查看和管理本门店的数据
- ⚠️ 导入的人员会自动关联到当前门店
- ⚠️ 无法查看其他门店的数据

## ⚠️ 注意事项

1. **执行顺序**：必须先执行数据库脚本，再启动服务
2. **浏览器缓存**：务必清除浏览器缓存，否则可能看不到新菜单
3. **默认密码**：新导入的用户默认密码为123456
4. **审核流程**：新导入的人员默认状态为"待审核"，需要管理员审核后才能使用

## 🛠️ 故障排查

### 问题1：菜单不显示
**原因**：权限未配置或浏览器缓存未清除
**解决**：
1. 执行 `plan/auth_permission_insert.sql`
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 强制刷新（Ctrl+F5）

### 问题2：页面404
**原因**：后端Controller未正确加载
**解决**：
1. 检查IDE是否正确识别了新增的Java文件
2. 重新启动后端服务
3. 检查控制台是否有启动错误

### 问题3：导入失败
**原因**：Excel格式错误或必填字段为空
**解决**：
1. 确保使用.xlsx格式的Excel文件
2. 确保员工工号和员工姓名必填
3. 不要使用合并单元格
