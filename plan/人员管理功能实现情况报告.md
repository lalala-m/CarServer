## 三、数据库表结构问题

### 3.1 缺失的字段
- **`store_id` 字段**：在所有相关表中均未找到此字段
  - `business_consultant` 表无 store_id
  - `info_user` 表无 store_id  
  - `financial_user` 表无 store_id

### 3.2 需要修改的表
```sql
-- 需要为以下表添加 store_id 字段
ALTER TABLE business_consultant ADD COLUMN store_id INT DEFAULT 0 COMMENT '所属4S店ID';
ALTER TABLE info_user ADD COLUMN store_id INT DEFAULT 0 COMMENT '所属4S店ID';
ALTER TABLE financial_user ADD COLUMN store_id INT DEFAULT 0 COMMENT '所属4S店ID';
```

## 六、技术实现要点

### 6.1 后端权限控制
```java
// 在BaseService中添加store_id过滤
@Override
protected void beforeGetList(JSONObject params) {
    // 获取当前用户
    User currentUser = getCurrentUser();
    if ("4S店管理员".equals(currentUser.getUserGroup())) {
        // 添加store_id过滤条件
        params.put("store_id", currentUser.getStoreId());
    }
}
```
