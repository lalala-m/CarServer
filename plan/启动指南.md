## 🚀 启动步骤

### 方式一：使用一键启动（推荐）

1. **确保Redis在WSL中运行**
   打开WSL终端，执行：
   ```bash
   redis-server --daemonize yes
   ```

2. **双击运行**
   ```
   project/一键启动全部.bat
   ```

### 方式二：手动启动

#### 1. 启动Redis（WSL中）

打开WSL终端：

```bash
# 启动Redis
redis-server --daemonize yes

# 确认Redis运行
redis-cli ping
# 应该返回: PONG
```

#### 2. 启动Spring Boot

打开PowerShell或CMD：

```bash
cd d:\AllProject\车友服务系统\project
mvn spring-boot:run
```

## 🔧 常见问题

### 问题1：Redis连接失败

**症状：** `Cannot connect to Redis`

**解决：**
```bash
# 在WSL中重启Redis
redis-cli shutdown
redis-server --daemonize yes
redis-cli ping
```

### 问题2：Maven未找到

**症状：** `'mvn' 不是内部或外部命令`

**解决：**
1. 下载Maven: https://maven.apache.org/download.cgi
2. 解压到 `D:\Tools\apache-maven`
3. 添加到系统环境变量 `PATH`

### 问题3：端口被占用

**症状：** `Port 5000 is already in use`

**解决：**
```bash
# 查找占用端口的进程
netstat -ano | findstr :5000

# 结束进程（替换PID）
taskkill /PID 1234 /F
```

### 问题4：MySQL连接失败

**症状：** `Access denied` 或 `Connection refused`

**解决：**
1. 确保MySQL服务运行中
2. 检查用户名密码是否正确
3. 检查数据库 `project_db` 是否存在
